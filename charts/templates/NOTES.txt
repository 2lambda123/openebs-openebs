
Successfully installed OpenEBS.

Check the status by running: kubectl get pods -n {{ .Release.Namespace }}

The default values will install LocalPV Hostpath Provisioner, LocalPV LVM, LocalPV ZFS and Mayastor and enable these
storage engines along with their default StorageClasses. The LocalPV offer non replicated local storage while Mayastor provides replicated storage.
Use `kubectl get sc` to see the list of installed OpenEBS StorageClasses.

For more information, 
- view the online documentation at https://openebs.io/docs or
- connect with an active community on Kubernetes slack #openebs channel.
